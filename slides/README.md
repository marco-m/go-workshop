# Go workshop slides

This directory contains already the slides in PDF format.

To regenerate them:
- Install [typst](https://github.com/typst/typst) at least 0.8.0
- Run `task slides`

To edit and automatically update the PDF:

```
typst watch go-workshop.typ
```
